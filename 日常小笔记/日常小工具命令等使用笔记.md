## DOCX 转 MD（Pandoc）

```bash
# 安装
scoop install pandoc  # Windows，Mac 用 brew install pandoc

# 单文件转换（GFM 保留表格，extract-media 导出图片）
pandoc input.docx -o output.md --from docx --to gfm --wrap=none --extract-media=./media

# 批量转换
for f in *.docx; do
  pandoc "$f" -o "${f%.docx}.md" --from docx --to gfm --wrap=none --extract-media=./media
done
```

- 表格要完整 → 用 `--to gfm`，不要用 `--to markdown`
- 图片要导出 → 必须加 `--extract-media=./某文件夹`
- 合并单元格 → Pandoc 不支持，需手动处理

### 其他常用参数

| 参数 | 作用 |
|------|------|
| `--standalone` / `-s` | 输出完整文档（含头部元数据） |
| `--toc` | 自动生成目录 |
| `--toc-depth=3` | 目录深度，默认 3 级 |
| `--metadata title="标题"` | 设置文档标题 |
| `--shift-heading-level-by=-1` | 调整标题级别（-1 全部降一级） |
| `--reference-doc=template.docx` | 用模板文件控制输出样式 |
| `--dpi=300` | 图片导出分辨率 |
| `--markdown-headings=atx` | 标题用 `#` 风格（不用下划线） |
| `--wrap=preserve` | 保留原文换行，不自动折行 |
| `-f docx -t plain` | 只要纯文本，不要格式 |
| `--list-extensions` | 查看支持的格式列表 |

### 常用输出格式（`--to` / `-t`）

```bash
pandoc input.docx -o output.md    -t gfm          # GitHub Markdown（推荐）
pandoc input.docx -o output.md    -t markdown     # 标准 Markdown
pandoc input.docx -o output.html  -t html         # 转 HTML
pandoc input.docx -o output.pdf   --pdf-engine=xelatex  # 转 PDF（需装 LaTeX）
pandoc input.docx -o output.tex   -t latex        # 转 LaTeX
pandoc input.docx -o output.rst   -t rst          # 转 reStructuredText
```
