### 二、 ⚙️ 虚拟机关键配置 (Configuration)

在创建虚拟机时的“自定义硬件”环节，按以下标准配置最流畅：

| 配置项 | 推荐参数值 | 说明 (Why) |
| :--- | :--- | :--- |
| **内存 (Memory)** | **4GB (4096MB)** | 2GB会卡，8GB更佳(前提是你电脑有16G内存)。不要超过物理机内存的一半。 |
| **处理器 (CPU)** | **2 处理器 / 2 内核** | 总核心数 = 4核。这样跑编译或开网页不卡顿。 |
| **硬盘 (Disk)** | **30GB - 50GB** | 默认20GB很容易满。选 **“拆分成多个文件”** (方便移动拷贝)。 |
| **网络适配器** | **NAT 模式** | **最省心**。只要宿主机(你的电脑)能上网，虚拟机就能上网。 |
| **CD/DVD** | **使用 ISO 映像文件** | 这里必须指向你下载的那个 Ubuntu `.iso` 文件。 |
| **USB控制器** | **USB 3.1** | 保证你插U盘进虚拟机时传输速度够快。 |

---

### 三、 ⚡️ 避坑小贴士

1.  **关于安装位置**：不要把虚拟机安装在 C盘！在 D盘/E盘 新建一个文件夹叫 `VM_Ubuntu`。
2.  **关于虚拟化报错**：如果启动报错提示 "Intel VT-x"，需要重启你的物理电脑，进入 BIOS 开启 **Virtualization Technology (虚拟化技术)**。
3.  **安装中途断网**：安装 Ubuntu 过程中，建议先**断开网络** (拔网线或断WiFi)，否则它会尝试在线下载更新，导致安装过程卡在 "Retriving file..." 很久。装好系统后再联网更新。
